\textbf{\color{olkSlate}OLK — Open Library Kashmir: The Complete Technical Manual}

First edition — compiled July 2026.

This document describes the OLK codebase as of commit \texttt{5a98f62} (\emph{``Redesign the requests page as clear per-exchange cards''}). Code listings are reproduced verbatim from the project source under \pathname{src/}, \pathname{supabase/}, and configuration files at the repository root; file paths are given relative to the repository root throughout.

The manual is intended as a living document. As the codebase evolves, keep it in sync: when you add a migration, a page, or a server action, add or update the matching section here in the same pull request. A manual that drifts from the code it describes is worse than no manual at all — see Chapter~21, \emph{The Maintenance Playbook}, for the house rule on this.
